# Skill: FYTNEZZ Shopify Theme

Use esta skill cuando el usuario pida crear, modificar, revisar o publicar cambios del tema Shopify de FYTNEZZ.

## Proceso

1. Leer `AGENTS.md` y `CODEX.md`.
2. Identificar si el cambio afecta `sections`, `snippets`, `assets`, `templates`, `config`, `layout` o `locales`.
3. Buscar archivos relacionados antes de editar.
4. Editar el minimo necesario.
5. Validar con `shopify theme check` si Shopify CLI esta disponible.
6. Resumir en espanol:
   - archivos modificados
   - objetivo del cambio
   - pruebas realizadas
   - riesgos o datos pendientes

## Precauciones

- No publicar al tema live sin aprobacion.
- No borrar contenido de `settings_data.json` sin confirmacion.
- No reemplazar Dawn completo si solo se necesita una seccion.
- No asumir que GitHub o Shopify ya estan autenticados.
